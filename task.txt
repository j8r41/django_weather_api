Targeted at:
- Basic knowledge of Python;
- Work with APIs;
- Working with HTML and CSS.

Task:
Create a simple web application for displaying information about the current weather in a specified city. It should have the following functionality:
- The user should be able to enter the name of the city in a search form.
- The application should use an API to retrieve weather data for the specified city from a service (e.g., openweathermap.org).
- The application should display the current weather in the specified city, providing information on temperature, humidity, pressure, wind direction and speed.
- The design of the application should be visually appealing and user-friendly. You can use HTML, CSS, and JavaScript to create the user interface.
- If possible, you can provide information on weather forecasts for the next few days.

Evaluation of the task is subjective and will depend on how well the candidate solves the set tasks, as well as how accurately and quickly the task is completed.

Направленное на: 
- Знание основ Python;
- Работа с API;
- Работа с HTML и CSS.

Задание:
Нужно создать простое веб-приложение для отображения информации о текущем погоде в заданном городе.
Оно должно иметь следующую функциональность:
- Пользователь должен иметь возможность ввести название города в форму для поиска.
- Приложение должно использовать API, чтобы получить данные о погоде в заданном городе из некоторого сервиса (например, openweathermap.org).
- Приложение должно отображать текущую погоду в заданном городе, предоставляя информацию о температуре, влажности, давлении и направлении и скорости ветра.
- Дизайн приложения должен быть красивым и понятным пользователю. Для создания пользовательского интерфейса можно использовать HTML, CSS и JavaScript.
- Если возможно, то дополнительно можно предоставить информацию о прогнозе погоды на следующие несколько дней.

Оценка выполнения задания субъективна и зависит от того, насколько хорошо кандидат решит поставленные задачи, а также от того, насколько аккуратно и насколько 
быстро он выполнит задание.
