Направленное на: 
- Знание основ Python;
- Работа с API;
- Работа с HTML и CSS.

Задание:
Нужно создать простое веб-приложение для отображения информации о текущем погоде в заданном городе.
Оно должно иметь следующую функциональность:
- Пользователь должен иметь возможность ввести название города в форму для поиска.
- Приложение должно использовать API, чтобы получить данные о погоде в заданном городе из некоторого сервиса (например, openweathermap.org).
- Приложение должно отображать текущую погоду в заданном городе, предоставляя информацию о температуре, влажности, давлении и направлении и скорости ветра.
- Дизайн приложения должен быть красивым и понятным пользователю. Для создания пользовательского интерфейса можно использовать HTML, CSS и JavaScript.
- Если возможно, то дополнительно можно предоставить информацию о прогнозе погоды на следующие несколько дней.

Оценка выполнения задания субъективна и зависит от того, насколько хорошо кандидат решит поставленные задачи, а также от того, насколько аккуратно и насколько 
быстро он выполнит задание.
